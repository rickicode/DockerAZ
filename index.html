üìã Kebutuhan Sistem

Pastikan server Anda memenuhi kebutuhan berikut sebelum instalasi

computer Sistem Operasi

  • Ubuntu 20.04 / 22.04 LTS
  • Debian 11 atau lebih baru
  • CentOS 8 / Rocky Linux 8
  • Atau distro Linux modern lainnya

memory Hardware Minimum

  • CPU: 1 Core (2+ direkomendasikan)
  • RAM: 1GB (2GB+ direkomendasikan)
  • Disk: 10GB space kosong
  • Network: Koneksi internet stabil

settings Software

  • Docker (diinstall otomatis)
  • Docker Compose v2+
  • curl atau wget
  • sudo access / root

router Network (Opsional)

  • Port 3012 untuk dashboard
  • Port 80/443 untuk Traefik
  • Domain untuk SSL otomatis
  • Cloudflare account (tunnel)
